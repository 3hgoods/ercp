내시경역행담췌관조영술은 흔히 ERCP (Endoscopic retrograde choangiopancreatography)로 불리우는 시술입니다. 일반적인 위·대장내시경과 달리 십이지장경과 방사선투시기를 이용하여 십이지장 유두부를 통해 담관 및 췌관을 조영하고 검사하는 시술


## ERCP 시행
ERCP는 담관 및 췌관과 연관된 질환이 의심되거나 발생한 경우 시행
